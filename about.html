Some Info

Moving

You move in Some Chess by clicking first on the square you want to move from and then on the square you'd like to move to.

Castling

The rule when castling is that you always move your king first. In Some Chess you move you king first and the rook will automaticaly go to the proper square.

Pawn Promotion

Once you get your pawn to the other player's back line you will be given the choice of what to promote the pawn to.

Stalemate

Right now Some Chess doesn't know what a stalemate is. The honor system is nice, both players should agree to draw (which is what a stalemate is).

Resign

If you want to give up click the "Resign" button. The game will be end and it will be counted as a loss for you. Be careful because there is no way to undo this.

Draw

You can suggest a draw at anytime by clicking on the "Draw" button. Your opponent must accept, otherwise the game will continue. To reject a draw offer all you have to do is make your next move.

Undo

You can request an undo within thiry seconds of your last move. Your opponent must accept, otherwise the game will continue. To reject an undo request all you have to do is make your next move. (Note: this function must be enabled by an admin)

Ending a Game

If a player hasn't made a move in a long time you will get the option to end the game. To end a game click on the game in the menu, there will now be a "End" button in the options panel under the chess board. If there are two or more complete moves in the game the win will go to the person who moved last, otherwise the game will simply be deleted. Be careful because there is no way to undo this. Note, the default for the number of days before the "end" option appears is twenty, but the admin can set this number higher or lower.

Points

Points are calculated using the standard system, the number of wins plus the number of draws divided by two or Wins + (Draws/2) = Points.

Notation

Some Chess uses pgn notation, which you can export once the game is complete. This is a nice way to save & play through your games again in another program, etc.

Importing PGN

To view a past game that is not in the database, played on Some Chess or elsewhere, all you need to do is paste the entire contents of the game into the field under the "Import Game" option.

Adding a User Image

If this option is enabled you can add a custom image to your account. The option is located in the "Profile" window of the "Menu" page, just click "browse" and locate any image file. Please note that the image will be automatically resized to 85x85 pixels. The following image formats are accepted PNG, GIF, and JPG. (Note: this function must be enabled by an admin)

Gonna Forget Your Password?

In order to reset your password you'll need to put in a security question and answer, otherwise you'll have to ask an admin to do it manually.
